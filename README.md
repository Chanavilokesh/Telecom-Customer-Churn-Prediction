# 🚀 Telecom Customer Churn Prediction

## 📌 Project Overview
This Machine Learning project predicts customer churn in the telecom industry using classification algorithms. The goal is to identify customers who are likely to leave the service so that the company can take proactive retention actions.

---

## ❗ Problem Statement
Telecom companies face significant revenue loss due to customer churn. The objective of this project is to build a predictive model that can identify potential churn customers in advance.

---

## 📊 Dataset Information
The dataset contains customer details such as tenure, contract type, monthly charges, payment method, and service usage patterns used to predict churn behavior.

---

## 🤖 Machine Learning Models Used
- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Gradient Boosting Classifier  
- XGBoost Classifier  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  

---

## 🏆 Best Model
Logistic Regression was selected as the best-performing model based on balanced evaluation metrics.

---

## 📈 Performance Metrics
- Accuracy: 82%  
- Precision: 82%  
- Recall: 82%  
- F1 Score: 82%  
- ROC-AUC Score: 91%
